## Описание проекта

CampusMadi — веб-приложение, предназначенное для просмотра расписания занятий и работы с заметками, привязанными к конкретным парам.


Проект минимально представляет собой веб-приложение для вуза с тремя ролями: студент, преподаватель и администратор.

Суть системы:

студент смотрит расписание своей группы и ведёт личные заметки к занятиям;
преподаватель смотрит расписание своих групп, добавляет заметки к своим парам и ведёт личные заметки;
администратор через отдельную админ-панель управляет пользователями, группами, преподавателями и самим расписанием.
По архитектуре это SPA-приложение с разделением на:

фронтенд;
бэкенд API;
базу данных;
